# 手續費

:::caution
回測訂單成交手續費 不會因交易所不同而改變
:::

### 手續費優先順序

- 讀取 [策略 option](/docs/strategy/python#存取策略參數) `exchange_fee` 來計算

- 預設用 0.001 計算

### 支付幣別優先順序

- 買入用 `target currency` 支付 / 賣出用 `base currency` 支付
