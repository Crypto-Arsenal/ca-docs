# Order

## order_id: integer

訂單編號

e.g. 1626072888367

## time: string

時間

e.g. 2021-01-05T08:01:00.000Z

## exchange: Exchange

交易所

e.g. Binance

## pair: ExchangePair

交易對

e.g. BTC-USDT

## type: OrderType

訂單類型

e.g. OrderType.LIMIT

## status: OrderStatus

訂單狀態

e.g. OrderStatus.FILLED

## price: float

價格

e.g. 25000

## amount: Exchange

數量

e.g. 1

## amount_filled: Exchange

成交數量

e.g. 0.5
