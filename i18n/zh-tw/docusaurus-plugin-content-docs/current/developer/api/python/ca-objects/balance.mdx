# Balance

### `exchange: Exchange`

**交易所** | 查看 [Exchange](/docs/developer/api/python/ca-objects/exchange)

e.g. Binance

### `currency: string`

**貨幣名稱**

e.g. BTC

### `available: float`

**可用資產**

e.g. 1626072888367

### `locked: float`

**鎖定資產**

e.g. 0.5

### `total: float`

**全部資產**

e.g. 1.5
