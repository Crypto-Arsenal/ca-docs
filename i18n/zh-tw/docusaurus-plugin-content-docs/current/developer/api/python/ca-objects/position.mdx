# Position

### `exchange: Exchange`

**交易所** 

e.g. Binance

### `pair: string`

**交易對名稱**

e.g. BTC/USDT

### `position_side: string`

**持倉方向**

e.g. LONG, SHORT

### `total_size: float`

**全部倉位**

e.g. 1.0

### `available_size: float`

**可用倉位**

e.g. 0.5

### `entry_price: float`

**開倉價格**

e.g. 1500

### `mark_price: float`

**標記價格**

e.g. 1500

### `margin: float`

**保證金**

e.g. 1500

### `liquidation_price: float`

**強平價格**

e.g. 100

### `profit_and_loss: float`

**未實現盈虧**

e.g. 10
