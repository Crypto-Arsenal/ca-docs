# get_orders

<Highlight color="#ffba00">
  {" "}
  僅可用於 trade() 或 on_order_state_change(){" "}
</Highlight>

## Input

```typescript
status: OrderStatus;
limit: number;
```

> 查看 [OrderStatus](/docs/developer/api/python/ca-objects/order-status)

## 回傳

```typescript
orders: [Order];
```

> 查看 [Order](/docs/developer/api/python/ca-objects/order)

## 範例

取得最新 5 筆已成交 (FILLED) 訂單。

```python
orders = CA.get_orders(CA.OrderStatus.FILLED, 5)
```

取得最新 3 筆新 (NEW) 訂單。

```python
orders = CA.get_orders(CA.OrderStatus.NEW, 3)
```
