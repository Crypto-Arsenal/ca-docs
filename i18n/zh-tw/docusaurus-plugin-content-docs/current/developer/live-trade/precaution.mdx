# 注意事項

## 交易所

### Bitget

- 當使用者為帶單者時只能以[市價](https://www.bitget.com/zh-TW/support/articles/8605995455257-Bitget%E4%B8%80%E9%8D%B5%E8%B7%9F%E5%96%AE-%E4%BA%A4%E6%98%93%E5%93%A1%E5%B9%B3%E5%80%89)平倉，也無法部分平倉。
- 不建議在 `place_order` 中使用 `profit` 及 `loss`。
