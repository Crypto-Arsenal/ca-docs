# OKX Testnet

Log in to the OKX exchange, click [Trade] -> [Demo Trading]


Click [Settings (the second icon on the right of Trading/Market Information)]


Click the first [Account Mode]


Set to [Single Currency Margin Mode]


In the upper right corner, click [User (the icon on the right side of asset management-simulated trading)] -> [simulated trading API]


Click [Apply for simulated trading V5 API] to create an API


Go to the CryptoArsenal Profile page, click [API Key] -> [manage]
